Quick tip:

Topic label is for downloading text based on the topic which contains whatever you want in topic form such as inflation. If you input inflation, the API will scrape from Twitter (X) every single comment/opinion about inflation.

And for count data, choose any number you want to scrape from the API (max 100 comments).

After you download the data, please add a column right after the text column named "sentiment". The class is 1 for positive, 0 for neutral, or -1 for negative. You can use number or text for sentiment class.
